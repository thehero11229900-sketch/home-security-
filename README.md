# 3. Home Safety Management System

## 1. Introduction
An IoT-based smart home safety system for monitoring and controlling home security devices.

## 2. Purpose
To ensure real-time safety monitoring and automated emergency responses.

## 3. Scope
Includes sensors, mobile app monitoring, and automated alerts.

## 4. Functional Requirements

1. Motion detection and camera integration.  
2. Door/window monitoring.  
3. Smoke/fire detection alarm.  
4. Temperature/humidity monitoring.  
5. Water/gas leak detection.  
6. Smart lock control and logs.  
7. Emergency contact notifications.  
8. Remote monitoring via app.  

## 5. Non-Functional Requirements

- Alert latency < 5 seconds.  
- Camera live feed < 5 sec delay.  
- 24/7 monitoring.  
- Data encryption.  
- Battery backup support.  

## 6. Acceptance Criteria

- AC-1: Motion in Away mode → push notification.  
- AC-2: Smoke → alarm + notify 3 contacts.  
- AC-3: Water leak → auto shut-off + alert.  
- AC-4: Live camera < 5 sec latency.  
- AC-5: Temp > 35°C → AC auto-on.  
- AC-6: Unauthorized door access → alarm + recording.  

## 7. Compliance & Auditing

- Maintain security event logs.  
- Annual safety compliance audit.  
- GDPR/data privacy compliance.
